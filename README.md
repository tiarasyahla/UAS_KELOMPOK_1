Sistem rekomendasi makanan sederhana berbasis Python.  
Proyek ini dibuat sebagai bagian dari Ujian Akhir Semester mata kuliah *Dasar Pemrograman*.

Aplikasi menyediakan fitur:
- Registrasi dan login
- Input preferensi makanan
- Rekomendasi makanan menggunakan library **faker-food**
- Profil pengguna (data diri, preferensi, achievement)
- Riwayat makanan / catalog
- Sistem achievement sederhana

Semua data disimpan **in-memory**, tanpa database eksternal.
